# insta-followers-diff — bookmarklet

> **THIS IS NOT THE RECOMMENDED WAY TO USE INSTA-FOLLOWERS-DIFF.**
> The console-paste path (see below) is the primary, recommended approach.
> See the [web app](../web/index.html) for the safe, zero-risk path.

---

## Why console paste is the primary, recommended path

Pasting the full, human-readable `console-tool.js` directly into DevTools lets you
read **exactly what the code does before running it**. The bookmarklet below is the
verbatim `console-tool.js` source, just URL-encoded — functionally identical but
harder to visually audit. The bookmarklet also loads **no remote code** (unlike
bookmarklets that `eval` a fetched script, which would be blocked by Instagram's
`script-src` CSP and would constitute a supply-chain risk). This bookmarklet is
self-contained — the same reviewed source, URL-encoded — but a long `javascript:`
URL is difficult to eyeball for tampering, so the paste path stays primary.

Some browser or CSP configurations may also interfere with bookmarklets; the
console path is the most reliable.
